#Solvable Games

Features some examples of puzzles solved by the selection monad in Haskell.
Most of the sequentially solved puzzles are Sudoku variations.
Solved in parallel:
  * Sudoku
  * Connect 3 / Connect 4
  * TicTacToe
  * NQueens

Computed sequentially:
  * Futoshiki
  * Nonograms
  * Buchstabensalat
  * Inshi no Heya
